# Install

## create links
dotfilesLink.sh

## vim bundle
* git submodule init
* git submodule update
* vim => :NeoBundleInstall

## add giftconfig
vim .gitconfig.local

> [user]  
>   name = "Your Name"  
>   email = you@example.com

## Uninstall
removeDotfilesLink.sh
